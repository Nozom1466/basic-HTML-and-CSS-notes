### How to display ?

- HOW to make the spreading ?
- set the hidden part to `display:none` 
	- default status
- give the open class a new class name then find the pseudo class and set the display to `display: block`
```css
/* DEFAULT STATE*/
.hidden-box {
	grid-column: 2;
	display: none;
}

.number,
.text {
	font-size: 24px;
	font-weight: 500;
	/* color: #087f5b; */
}

.number {
	color: #ced4da;
}




/* OPEN STATE */
.open {
	border-top: 4px solid #087f5b;
}

.open .hidden-box {
	display: block;
}

.open .number,
.open .text {
	color: #087f5b;
}
```

### end selector

- we want to give special css to the child element, we usually give the special one another class
- in which case if we only select the new-added selector, like the one below

```html
<a class="page-link page-link--current">CONTENT</a>
```

```css
.page-link:link {           /*(0, 2, 0)*/
	color: #000;
}

.page-link--current {       /*(0, 1, 0)*/
	color: #fff;
}
```
- then the CONTENT would be `#000` because the priority of `.page-link` is higher than that of `.page-link--current` (noted in the code)





- end selector is used to select the labels that exactly has the same class type (number and type) 
```html
<a class="page-link page-link--current">CONTENT</a>
```

```css
.page-link:link {                     /*(0, 2, 0)*/
	color: #000;
}

.page-link.page-link--current {       /*(0, 2, 0)*/ /*no space between two selectors*/
	color: #fff;
}

```
>now the two selectors has the same priority
>the CONTENT would be `#fff`

## Hero Section Components

- `<header>`, `<nav>` sematic label


### centering the object

- create a class then add the class to the labels
```css
/*center the object*/
.container {
	margin: 0 auto;
	width: 1200px;       /*designate the width*/
}
```


### Unit vh

- `1vh` is the height of view height
- usually is set to 100 or less (in percent)
```css
header {
	height: 100vh;    /*occupy the screen*/
}
```



### backgound-image

- set image
```css
header {
	background-image: url(hero.jpg);
	background-size: cover;           /*adjust the size tp view size*/
}
```
>wrapped in `url()`



- set daker cover over background-image
- use backgroud-image **STACK**
```css
header {
	background-image: linear-gradient(
		rgba(34, 34, 34, 0.6),
		rgba(34, 34, 34, 0.6)
	),
	url(hero.jpg);
	background-size: cover;
}
```
>use a gradient cover over original image


- other applications of gradient
```css
nav {
	background-image: linear-gradient(to right, red, blue); 
}
```
>gradient from left to right

### height of body

- the body will not occupy the view automatically

```css
body {
```

### how to set fr unit ?

- shrink the page to see if certain part is flexiable or fixed
- fixed : `XXpx`
- flexiable : `XXfr`


### about button

- `<button>` should be set to `inline-block` 
- always set border of button to none
```css
button {
	display: inline-block;
	border: none;
	cursor: pointer;
}
```

```css
}
```



### Create a slider 

![image.png](https://sandox-1316371247.cos.ap-shanghai.myqcloud.com/ObsidianImg/202301242338560.png)


- in the parent class : set `overflow` to `scroll`
- in the child class : set flex-shrink to 0
	- the flexbox would shrink the elements in the flexbox

```css
section {                     /*parent class*/
	overflow: scroll;

	display: flex;
	flex-direction: column;
	gap: 40px;
}

.email {
	flex-shrink: 0;          /*no shrinking*/

	display: flex;           /*for aligning text in the box*/
	align-items: center;
	justify-content: center;
}

```
